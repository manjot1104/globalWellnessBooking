# Assets folder

## Hero video
Place the hero background video here as **`hero-video.mp4`**.

If your file is named something else (e.g. "grok something.mp4"), rename it to `hero-video.mp4` when you move it here, or update the path in `components/sections/Hero.tsx` (`HERO_VIDEO_SRC`).

## Service videos
Optional videos for the **Our Services** section (homepage and `/services` page). Place one file per service with these exact names:

| Filename | Service |
|----------|---------|
| `anxiety.mp4` | Anxiety Counselling |
| `depression.mp4` | Depression Counselling |
| `relationship.mp4` | Relationship Counselling |
| `stress.mp4` | Stress Management |
| `child.mp4` | Child Counselling |
| `career.mp4` | Career Counselling |

If a video is missing or fails to load, the service card falls back to the default icon.
