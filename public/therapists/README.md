## Folder Structure

Place therapist photos in this folder with the following naming convention:
- Use the therapist's email (without @domain.com) as the filename
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- Recommended size: 400x400px or larger (square format preferred)

## How It Works

The application will automatically:
1. Look for a photo file matching the therapist's email
2. If found, display the photo
3. If not found, display a circle with the therapist's initial

## Adding Photos

1. Save the photo file in this folder (`client/public/therapists/`)
2. Name it using the therapist's email (e.g., `kumarsunil1002452.jpg`)
3. The photo will automatically appear in the application

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Size**: Minimum 400x400px (square recommended)
- **File Size**: Keep under 500KB for faster loading
- **Quality**: Clear, professional headshot or portrait
